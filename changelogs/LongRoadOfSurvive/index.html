LongRoadOfSurvive

BEAT+Build.7

机制

  • 镍在地底世界每个矿脉最大矿物生成数量为8,主世界也为8
  • 镍在地底世界每个区块最大矿物生成数量为15,主世界也为8
  • 在沼泽和针叶林也生成绿宝石

BEAT+Build.6

方块

  • 石高炉速度提升为50%
  • 熔岩高炉速度提升为150%
  • 下届高炉速度提升为200%
  • 添加了烟熏炉速度提升100%

机制

  • 烟熏炉只能放食物
  • 高炉只能放方块
  • 镍武器无法攻击火元素
  • 镍矿燃烧获得30Exp

BUG

  • 修复了高炉无法加速的bug

BEAT+Build.5

方块

  • 完善了国际化和合成配方(如果不能合成,看合成表)

物品

  • 完善了国际化和合成配方(如果不能合成,看合成表)

BEAT+Build.4

方块

  • 完善了附魔回流底座

附魔回流底座

  • 上方可以放提供经验的宝石
  • 下方可以放药水(包括水瓶)或镍粒
  • 药水生成附魔之瓶,镍粒生成镍币

机制

  • 所有宝石不再直接提供经验

BEAT+Build.3

物品

  • 完善了镍工具
  • 添加了一点配方
  • 国际化镍工具了一点
  • 镍币提供250EXP

方块

  • 添加了附魔回流底座

附魔回流底座(测试方块)

概述：一种特殊的远古金属铸造的回路可以让那些“力”在里面流转这有助于我们利用那本应无法利用的宝石
  • 上方放钻石,目前仅支持钻石
  • 下方放药水(包括水瓶),先放钻石再放水瓶,输出附魔之瓶

材质都是Creespic提供的啦

BEAT+Build.2

物品

  • 添加了一系列镍工具(材质由Creespic提供)

机制

  • 镍与铁同级
  • 镍全套时免疫一切史莱姆伤害和腐蚀
  • 镍工具=附魔工具

方块

  • 添加了镍块,镍砧
  • 添加了石高炉,熔岩高炉,下界高炉依次提高燃烧速度(材质由Creespic提供)
  • 高炉仅用来冶炼矿物,且烧炼的物品和燃料热度必须相同或燃料热度大于烧炼的物品热度时拥有速度加成
  • 速度加成是多少?我不到啊

bug
